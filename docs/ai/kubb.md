# Kubb AI Context

When working with Kubb in this repo, use the local `kubb` skill plus Kubb's official 5.x AI documentation as the source of truth.

- MCP docs: https://kubb.dev/docs/5.x/ai/mcp
- Skills docs: https://kubb.dev/docs/5.x/ai/skills
- LLM docs index: https://kubb.dev/docs/5.x/ai/llmstxt
- Installed local skill: `.agents/skills/kubb/SKILL.md`
- Local MCP command: `npx kubb mcp`

The official Kubb skills page currently says Kubb-specific AI coding skills are a placeholder. This repo-local skill keeps the workflow available until official Kubb skills are published.
